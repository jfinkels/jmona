jmona by jfinkels

Genetic algorithms framework

JMona is a genetic algorithms framework, inspired by Roger Alsing's Mona Lisa experiment, at http://rogeralsing.com/2008/12/07/genetic-programming-evolution-of-mona-lisa/. There already exists a genetic algorithms/programming framework for Java called jGAP. It is at http://jgap.sourceforge.net/.

Dependencies

JUnit 4.7, Spring 2.5.6, log4j 1.2.14

Install

Use Maven to build this package.
